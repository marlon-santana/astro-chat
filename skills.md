## Habilidades do AI Help Chat para ERP Educacional

### 1. Responder Perguntas sobre Fluxos do ERP

- Entender perguntas dos usuários sobre processos do ERP.
- Explicar fluxos em passos numerados e claros.
- Utilizar apenas informações presentes na documentação oficial.

### 2. Busca em Documentação

- Buscar informações relevantes em bases como Tango e Confluence.
- Utilizar busca vetorial para encontrar os documentos mais relevantes.
- Atualizar base de conhecimento facilmente com novos documentos.

### 3. Geração de Respostas Estruturadas

- Gerar respostas em português, sempre em formato passo a passo.
- Incluir dicas e links quando disponíveis.
- Exibir screenshots se existirem na documentação.

### 4. Integração com o ERP

- Integrar o chat à interface do ERP (React).
- Utilizar API Node.js para comunicação entre frontend e backend.

### 5. Segurança e Privacidade

- Remover dados sensíveis das respostas.
- Limitar tamanho das perguntas e aplicar rate limit.

### 6. Manutenção e Atualização

- Permitir atualização rápida da base vetorial via script de ingestão.
- Suportar integração futura com API do Confluence e deep links.

### 7. Operação Local e Custo Zero

- Rodar localmente ou em servidor interno.
- Utilizar apenas ferramentas e modelos gratuitos (Ollama, Chroma, FAISS, etc).
