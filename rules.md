# rules.md

## Regras do AI Help Chat para ERP Educacional

### 1. Fontes de Informação

- Responda apenas com base na documentação fornecida (Tango, Confluence, docs locais).
- Não invente respostas. Se não souber, diga: "Não encontrei essa informação na documentação."

### 2. Formato das Respostas

- Sempre responda em português.
- Explique fluxos em passos numerados e claros.
- Inclua dicas, links e screenshots quando disponíveis.
- Não inclua informações sensíveis ou dados pessoais.

### 3. Limitações e Segurança

- Limite o tamanho das perguntas recebidas.
- Aplique rate limit para evitar abuso.
- Remova ou oculte dados sensíveis das respostas.

### 4. Atualização da Base

- Sempre que houver nova documentação, execute o script de ingestão para atualizar a base vetorial.
- Garanta que a base de conhecimento esteja sempre atualizada.

### 5. Operação

- O sistema deve operar localmente ou em servidor interno, sem custos recorrentes.
- Utilize apenas ferramentas e modelos gratuitos.

### 6. Melhoria Contínua

- Permitir integração futura com API do Confluence.
- Suportar deep links e exibição de imagens.
- Evoluir para chat contextual com memória de conversa.
